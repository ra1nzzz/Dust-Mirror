# DustMirror (观尘)

DustMirror release distribution repository.

## Download

Download the latest version from [Releases](https://github.com/ra1nzzz/Dust-Mirror/releases).

## Auto-Update

DustMirror checks for updates automatically on startup.

## Features

- Encrypted configuration storage
- OTA auto-update system
- PyWxDump community script integration
- PRO module support (requires sponsor code)
