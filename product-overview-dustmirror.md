# 观尘（DustMirror）产品概述

> 本文件用于公开发布仓库，介绍观尘的整体能力与版本分层。  
> 标注说明：**[PRO]** 表示需要赞助/授权后解锁的高级能力，未标注功能默认属于免费版可体验的核心流程。

## 产品定义

观尘（DustMirror）是一款本地优先的微信聊天数据洞察与数字分身构建平台，面向个人用户与轻团队。  
核心理念是：**隐私可控、证据可追溯、能力可导出**。

产品命名建议保留双层表达：
- 正式名：观尘
- 英文名/品牌名：DustMirror
- 技术仓库名：可继续沿用内部仓库名，对外宣传统一使用“观尘（DustMirror）”

## 一句话定位

**观尘：把个人聊天记录和博客文章、个人知识库沉淀为可检索、可审阅、可复用的个人知识与数字分身。**

## 产品现状

当前项目已具备较完整闭环，包括：

- 聊天数据导入（注：本产品不涉及任何逆向功能，由用户自主选择使用相关开源仓库进行导出）
- 消息、会话、联系人、总结、语料、画像等结构化存储
- 仪表盘、趋势、热力图、活跃群组、话题词云
- 消息浏览、@我消息、我发送的消息、会话与成员分析
- 每日群聊总结与 LLM 生成能力
- 语言画像、记忆体系、关系画像与图谱原型
- Skill / 语料导出能力
- API Key 访问控制与只读接口
- FastAPI 服务、任务系统、前后端一体化 SPA

## 核心架构

```text
前端 SPA
   ↓
FastAPI 主服务
   ├── 消息采集与增量同步
   ├── 数据仓库（SQLite / WAL）
   ├── 分析 API（仪表盘 / 消息 / 话题 / 联系人 / 会话）
   ├── 摘要与 LLM 模块
   ├── 数字画像与蒸馏引擎
   ├── 证据链与身份断言体系
   ├── 关系画像与知识图谱
   ├── 任务系统与作业队列
   └── 开放 API 与 API Key 管理
```

## 产品价值

观尘的核心价值不只在于“查看聊天记录”，而在于三层能力：

1. **沉淀**：把零散对话变成结构化资产  
2. **理解**：从消息中提炼关键话题、决策、待办与人物关系  
3. **复用**：把画像、语料、Skill、关系信息输出给后续系统使用

- 简单活跃时段分析

## 用户场景

### 个人用户
- 整理高频群消息，快速看重点
- 回顾最近联系最多的人、最活跃的群
- 构建自己的语言风格与表达偏好
- 回放关键事件、决策、待办

### 轻团队 / 运营用户
- 统计社群活跃度与关键发言者
- 沉淀客户沟通线索与上下文
- 通过受控 API 为外部系统提供只读数据
- 导出 SFT / Skill 供后续 Agent 训练与接入

## 产品路线图

1. 品牌统一：统一使用“观尘（DustMirror）”
2. 免费版打磨：降低首次使用门槛，稳定采集与浏览体验
3. PRO 能力封装：把深度画像、关系分析、导出、任务、API 升级为 PRO
4. 商业闭环：授权、校验、订阅、升级入口

## 结论

- 免费版用于让用户快速感受到“消息沉淀 + 基础理解 + 个人入口”
- PRO 版用于放大“数字分身 + 关系洞察 + Skill 复用 + 开放集成”的长期价值
